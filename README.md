# Future Family

Future Family is a single-page Next.js app for discovering elegant future-child names for a French couple.

## Run

```bash
npm install
npm run dev
```

Open the local URL printed by Next.js, usually `http://localhost:3000`.

## Notes

- No backend, no authentication, no external API calls from the app.
- Saved favorites, maybe names, rejected names, and discovery history are stored in `localStorage`.
- The offline catalogue builds to 2,713 boy entries and 2,626 girl entries from curated real-name seeds and traditional compound forms.
- Hidden Gems mode prioritizes rare, elegant, French-pronounceable names with good international compatibility.
